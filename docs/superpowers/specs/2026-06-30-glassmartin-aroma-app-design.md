# GlassMartin 空间香氛服务 APP 设计方案

## 目标

为 GlassMartin 澳洲品牌空间香氛服务制作一款客户展示与询价用的手机端网页 APP。客户可以按商业场景浏览推荐香型、参考区间价和扩香机型号图片，选择精油升数和扩香机数量后生成询价清单，并通过复制文本或生成图片发送给 GlassMartin 确认最终报价。

第一版重点是轻量、可转发、可添加到手机主屏幕、便于后续维护。不做登录、后台管理、数据库和最终自动锁价。

## 品牌与视觉方向

- 品牌名：GlassMartin
- 品牌副标题：澳洲品牌 · 空间香氛服务
- 顶部展示用户提供的 GlassMartin logo。
- 视觉方向：商业方案顾问型，带一点高端酒店香氛气质。
- 页面重点服务酒店、娱乐、商业采购客户，信息表达要专业、清楚、适合手机端快速选型。

## 核心客户流程

1. 客户打开网页 APP。
2. 系统先弹出近期大促弹窗，展示满指定金额/升数赠送 GAS-501F 扩香机的活动信息。
3. 客户关闭弹窗或点击查看推荐方案后，进入首页。
4. 首页展示品牌、商业场景入口和热门标准方案。
5. 客户选择商业场景，例如酒店、KTV / 酒吧 / 台球 / 网咖、会所 / SPA / 足浴等。
6. 系统展示该场景下推荐/常备香型、参考区间价和推荐扩香机。
7. 客户选择精油升数、扩香机型号和数量。
8. 选择内容进入询价清单。
9. 客户复制清单文字或生成清单图片，发送给 GlassMartin 确认最终报价。

## 页面结构

### 首页

首页顶部显示 GlassMartin logo、品牌名和“澳洲品牌 · 空间香氛服务”，右侧显示询价清单入口。首页主入口是“选择商业场景”，上方放可自动滚动的品牌广告 banner，下面放“热门标准方案”，用于承接常备库存和快速询价。

首页场景建议包含：

- 酒店 / 大堂 / 走廊
- KTV / 酒吧 / 台球 / 网咖
- 会所 / SPA / 足浴
- 售楼处 / 展厅
- 办公室 / 商务空间
- 商场 / 零售门店
- 卫生间 / 除味空间

### 场景推荐页

客户选择场景后，页面展示该场景的推荐方案。推荐香型优先展示常备/推荐香型，其他香型放在“更多可询香型”中。

香调不作为主筛选条件，只作为详情里的辅助信息。这样可以减少客户因不懂香调而误选，也更利于备货管理。

### 产品详情/选择区

客户可以选择精油升数，也可以选择扩香机型号和数量。价格只显示参考区间价或阶梯价，不显示最终锁定总价。

当询价清单内精油总升数每满 3L 时，系统提示可赠送 1 台 GAS-501F 插电蓝牙 APP 款扩香机。例如：

- 3L：赠送 1 台
- 6L：赠送 2 台
- 9L：赠送 3 台

赠送数量计算规则：

```js
giftMachineCount = Math.floor(totalOilLiters / 3)
```

赠送内容在清单中显示为“满赠扩香机：GAS-501F 插电蓝牙 APP 款 × N 台”。最终赠送和报价仍以 GlassMartin 确认为准。

### 询价清单

询价清单汇总客户选择的商业场景、香型、精油升数、扩香机型号、扩香机数量、满赠扩香机数量和备注。

清单支持：

- 复制文本，适合客户发微信。
- 生成图片，带 GlassMartin logo，适合客户保存到相册后发微信。
- PDF 作为后续增强功能，不进入第一版重点。

清单中不做最终总价锁定，只显示参考区间价和“最终报价以 GlassMartin 确认方案为准”。

## 价格策略

价格采用“参考区间价 + 询价确认”。

客户看到不同采购量对应的价格区间，例如：

- 1-3L：参考价
- 4-9L：参考价
- 10L+：参考价

具体区间可在数据文件中配置。最终报价根据采购数量、香型、机器配置、库存和活动规则由 GlassMartin 确认。

## 扩香机策略

扩香机采用“场景推荐 + 可加入询价清单”。

每个场景可配置推荐机器。客户也可以手动选择型号和数量。机器信息包含图片、型号、适用面积、电源方式、蓝牙/APP 等卖点和适用场景。

GAS-501F 插电蓝牙 APP 款作为默认满赠机型。

## 数据维护方式

第一版采用“Excel 表格 -> 导出数据文件”的维护方式。

用户维护 Excel 表格，开发侧提供导出脚本或导入流程，把表格转换为 APP 使用的本地数据文件，例如 `products.json`。

建议拆分或包含以下数据表：

- `scenes`：商业场景、排序、展示文案。
- `scents`：香型名称、适用场景、是否推荐/常备、参考区间价、描述、香调辅助信息、可询状态。
- `machines`：扩香机型号、图片文件名、适用面积、卖点、适用场景、是否推荐、是否可作为赠品。
- `packages`：热门标准方案，包含场景、推荐香型、建议升数、推荐机器和数量。
- `banners`：首页滚动广告图，包含标题、副标题、图片路径、跳转目标、排序和启用状态。
- `promotions`：活动弹窗文案、活动开关、赠送规则、活动有效期。
- `settings`：品牌信息、赠送规则、询价清单提示文案。

图片放在固定图片文件夹中，数据表中只填写图片文件名。这样后续新增、删除或修改香型与机型时，主要操作是改 Excel 和替换图片。

## 促销弹窗

单页 APP 支持弹窗。默认弹窗内容：

“近期大促：采购满指定金额/升数，可赠送 GAS-501F 插电蓝牙 APP 款扩香机。”

弹窗文案、是否启用、活动规则和展示按钮都放进数据文件中，方便后期修改。第一版可以在客户首次打开首页时展示一次，并允许关闭。弹窗操作按钮中，“先看看首页”优先展示，并使用黑色主按钮；“查看推荐方案”作为次级按钮。

## 手机保存与转发

APP 做成 PWA 网页 APP。客户打开公网链接后，可以添加到手机主屏幕，像普通 APP 一样使用。

只要部署到公网地址，客户可以把网页链接转发给其他客户。其他客户打开后也可以正常浏览图片、查看参考区间价、选择香型和机器，并生成询价清单。

如果只是本机或局域网地址，则外部客户不能访问。正式交付需要部署到公开访问地址。

## 第一版范围

第一版包含：

- 品牌首页
- 首页滚动广告 banner
- 场景筛选
- 推荐/常备香型展示
- 更多可询香型
- 参考区间价
- 扩香机图片和型号展示
- 精油升数选择
- 扩香机型号和数量选择
- 每满 3L 赠送 GAS-501F 的提示和清单计算
- 询价清单
- 复制文本
- 生成图片
- 促销弹窗
- Excel 到本地数据文件的维护流程
- PWA 添加到手机主屏幕

第一版暂不包含：

- 后台管理系统
- 登录账号
- 在线支付
- 自动锁定最终总价
- 多客户报价记录云端保存
- PDF 导出

## 验收标准

- 客户能在手机端打开并浏览首页、场景、香型、机器和参考区间价。
- 客户能按场景找到推荐/常备香型。
- 客户能选择精油升数和扩香机数量。
- 询价清单能正确汇总所选内容。
- 精油总升数每满 3L，清单中正确显示赠送 GAS-501F 数量。
- 客户能复制询价文本。
- 客户能生成带 GlassMartin 品牌信息的询价图片。
- 促销弹窗内容能从数据文件配置。
- 首页滚动广告内容能从数据文件配置。
- 产品资料能通过 Excel 导出数据文件维护。
- 网页部署后可转发给其他客户访问。
